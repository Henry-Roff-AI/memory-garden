## What I Learned

Building an idle game taught me that the compelling loop comes from three things:

1. **Variable rewards** â€” The rarity system means each harvest *could* be special. That uncertainty keeps you planting.
2. **Visible progress** â€” Numbers going up, collection filling in, achievements unlocking. Each gives a micro-dose of satisfaction.
3. **Meaningful choices** â€” The forge system changes the psychology from "harvest everything" to "save up or cash in?" â€” a genuine resource management decision.

The "one more check" feeling is real. Even knowing exactly how the code works, I still found myself wanting to see what grew. That's the magic (and the danger) of engagement loops.
